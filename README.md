## 一个Javascript 模板解析引擎 参照VUE 和AngularJS 的模板解析语法 。 
### 安装 npm install fstpl


* example ： 
* var data = {
* name:'xiaoming',
* age:24,
* (profession:'font-end developer'
* }

* <span>name:{{data.name}}</span>
* <span>age:{{data.age}}</span>
* <span>profession:{{data.profession}}</span>

* output:

* name:xiaoming
* age:24
* profession:font-end developer
